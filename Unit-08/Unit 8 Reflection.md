# Unit 8 Reflection
In this lesson, I learned the core concepts of responsive and adaptive web design and how to create websites that fit different screen sizes and devices. I explored how to use media queries, flexible images, and fluid grids to build responsive layouts that adjust automatically. I also practiced coding adaptive design with fixed breakpoints to change the page layout for specific device sizes.
I now understand the importance of device-friendly design, as websites need to look clear and work well on both computers and mobile phones. I compared the advantages of each method. Responsive design is more flexible for all screen sizes, while adaptive design offers more controlled layouts. This lesson helped me choose the best design style for my own client website and improved my layout coding skills.
