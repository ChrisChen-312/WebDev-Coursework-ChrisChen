3 Main Types of Cephalopods

  1. Octopus
  2. Squid
  3. Cuttlefish

Octopus

Key feature: 8 long arms

Octopuses are very intelligent and can solve problems.

Common Octopus Abilities:

  • Change color for camouflage
  • Squirt ink to escape enemies
  • Open jars and solve puzzles

Squid

Key feature: 8 arms + 2 tentacles

Squid swim fast and live in groups.



Cuttlefish

Fun fact: W-shaped pupils

Cuttlefish are not fish they are cephalopods!



Learn more from the Smithsonian Ocean website: Octopuses, Squids, and Relatives
