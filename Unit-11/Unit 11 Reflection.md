# Unit 11 Reflection
Looking back on my journey as a web designer, I can see how far I’ve come from writing basic HTML to building multi‑page, interactive websites. At first, I only thought about making pages look “nice,” but this unit taught me that good design is about solving problems and serving users. I learned to combine HTML, CSS, and JavaScript to create responsive layouts, consistent navigation, and even simple interactivity. I also studied core design principles like visual hierarchy, consistency, and color theory, which changed how I plan and build pages.
This experience has shown me that web design is not just about technical skills—it’s about thinking critically about user needs, accessibility, and purpose. I feel proud of the projects I’ve created, especially the cephalopods education site, which combined design with real‑world value. Moving forward, I want to keep learning more advanced JavaScript and responsive design to make even more engaging, inclusive websites. I know this is just the start of my growth as a designer, and I’m excited to keep challenging myself to build functional, stylish, and meaningful web experiences.
