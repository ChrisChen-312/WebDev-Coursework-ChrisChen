# Unit 4 Reflection
In this unit, I added three new features to my client’s webpage: lists, embedded images, and accessibility improvements. I found it really fun to see the page come together, with ordered and unordered lists making the information clearer, and an image adding visual interest to the content. Being able to build my own website from scratch feels exciting and satisfying.
At the same time, I also realized there are challenges. For example, making sure the image has a clear alt text for accessibility, and keeping the page well-structured for screen readers, required me to think more carefully about how different users will experience the site. These small details matter a lot, and I can see how even simple HTML requires careful planning.
Overall, I’m enjoying learning how to build and improve my own webpage, and I’m curious to keep adding more features as I go.
